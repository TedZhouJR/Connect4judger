## Connect-4 Judger
* A judger which can judge win/loss of any ongoing connect-4 game.
* Also a pytorch framework for future ML assignments, keep updating...

## Dataset
* UCI Connect-4 Data Set (http://archive.ics.uci.edu/ml/datasets/connect-4)
